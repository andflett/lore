# Naming candidates

> **Outcome (historical note):** the brand ultimately shipped as **Wyrdscribe**
> (`wyrdscribe.app`), which post-dates this shortlist — it isn't listed below.
> The doc first leaned toward **Hearthnote** (see "My picks"); that was a brief
> interim name. This page is kept for provenance, not as the current answer.

"Lorekeeper" is widely used (a Final Fantasy ability, several WoW guild
add-ons, multiple itch.io games, a Roblox simulator, etc.) — fine as a
working name during development, but worth rebranding before any public
launch. This doc lists candidates that match the dark-fantasy / scribe /
companion theme and pass a basic "is anyone else using this prominently?"
check.

For each candidate: a short note on connotation, what was checked, and
verdict (✅ candidate / ⚠️ collision worth knowing / ❌ avoid).

---

## Top tier (verified clear)

### Codexweave ✅
Codex (book of lore) + weave (binding playthroughs together). Reads as
"the weave of your codex." Unique on web search; no software collisions.
Domain `.com` is the obvious one to check; alternatives like `.app` /
`.gg` likely available.

### Bardspark ✅
A bard's spark of recall — small, ownable, modern. No npm/GitHub
collisions. Loose association: there's a `bard-ai` npm package (Google
Bard reverse-engineered API) — *different name*, no conflict.

### Hearthnote ✅
Hearth = home base / safe room; notes you keep at the fire. Warm and
gaming-flavoured without being on-the-nose fantasy. No collisions found.

### Glyphvault ✅
Glyphs (cryptic notes) + vault (locked records). More cryptic / darker
tone; suits the gilded-stone aesthetic. No collisions; some unrelated
"Glyph"-prefixed products (Glyphr Studio = font tool; Glyphbound =
Minecraft addon) exist but none clash on the full word.

### Vellumkeep ✅
Vellum (parchment) + keep (fortress). Carries the carved-stone feel
directly. No collisions found.

### Wayfarer's Ledger ✅
Wayfarer = journeyer; ledger = the record you keep. Slightly long for a
product name but descriptive and on-theme. No collisions found.

---

## Second tier (worth considering)

### Compendius ✅
Invented from "compendium" + Latinate ending. Reads as a name for a tool
that compiles things. Unique. Might feel too academic for some.

### Memoria Ludorum ✅
Latin "memory of games." Niche, specific, hard to mistake. Strong if the
audience leans TTRPG / classics-curious. Risky for broader appeal.

### Pilgrim's Codex ✅
Pilgrim = journeyer; codex = sourcebook. Slightly long; reads more
solemn than the others.

### Inkward ✅
Made-up. "Ink-ward" suggests a writer-as-protector, or "inward writing"
(memory). Short, ownable, distinctive. Unique.

### Recollect ⚠️
Verb. Reads cleanly as a tool ("Recollect — your gaming companion") but
the word is heavily used in marketing copy generally, and there's a
public Recollect.com (defunct? a data archive tool?). Worth a deeper
search before committing.

### Bookkeeper's Brand ✅
Playful; "brand" = mark. Quirky but charming. Probably too whimsical for
the dark-fantasy aesthetic.

---

## Tier 3 (drop)

### Tendril ❌
Already used by **Tendril Design + Animation**, a prominent motion-graphics
studio. Brand collision in the creative/design space; would confuse anyone
in that audience.

### Saga's Edge ❌
"Saga" is overused in tech ("Saga pattern" in distributed systems is a
loaded term). Likely many small collisions.

### Tabula Quest ❌
"Tabula" reads as an Atari mid-90s vibe; obscure enough but doesn't sing.

### Sidequest Codex ⚠️
"SideQuest" is the well-known sideloading tool for Quest VR. Too close
to that brand.

---

## My picks

If I had to recommend three to take forward:

1. **Codexweave** — best signal-to-noise; reads as exactly what the app does
2. **Hearthnote** — warmest, friendliest; less "lore" more "companion"
3. **Glyphvault** — strongest match for the carved-stone aesthetic

All three: clean search, no major collisions, easy to pronounce and spell,
likely-available domains, and they each scope differently (lore /
warmth / aesthetic) so the choice depends on what you most want the brand
to lead with.

## Checks done

- Web search for each candidate name + "app" / "software" / "github" / "npm"
- Manual scan of competing products in the gaming-companion / note-taking
  space (Codex Notes, SideQuest, Roll20, etc.)
- Brand-collision check for the creative/design space (Tendril)

Not checked (still to do before committing):

- Domain availability (`.com`, `.app`, `.gg`)
- Trademark search (USPTO, WIPO)
- App store name conflicts (iOS / Play)
- Social handles (X, Mastodon, GitHub org)
